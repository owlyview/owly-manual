00 · 오울리뷰 소개

아기 곁의 수호 부엉이에 오신 걸 환영해요

오울리뷰는 스마트폰 두 대만 있으면 바로 쓸 수 있는 무료 AI 아기 모니터 앱입니다.

📋 준비물

  • 안드로이드 스마트폰 2대 (서로 다른 기종 가능)
  • 두 폰이 같은 Wi-Fi에 연결되어 있어야 해요
  • Google Play에서 Owly View 검색 후 설치
  • 1
    📡 실시간 영상 모니터링
    Wi-Fi를 통해 허브폰의 카메라 영상을 뷰어폰으로 실시간 전송합니다. 별도 서버 없이 기기끼리 직접 연결돼요 (P2P).
  • 2
    😢 AI 울음 감지
    아기 울음소리를 AI가 실시간으로 감지해 뷰어폰에 알림을 보냅니다. 모든 분석은 기기 안에서만 이루어져요.
  • 3
    👶 AI 얼굴 감지
    아기 얼굴이 카메라에서 보이지 않으면 알림을 드려요. 뒤집히거나 자리를 이탈했을 때 바로 알 수 있어요.
  • 4
    🦉 육아 상자
    수유·수면·약 알림·성장 기록을 한 곳에서 관리할 수 있어요. 오울리뷰만의 육아 도우미 기능이에요.
  • 5
    🔒 완전 무료 & 개인 정보 보호
    모든 기능이 무료예요. 영상·음성은 외부 서버를 거치지 않아 개인 정보가 유출되지 않습니다.

야간 테마 배경

B
🌐 언어 전환 버튼 (우측 상단)
KO ↔ EN 버튼으로 앱 전체 언어를 바꿀 수 있어요. 역할 선택부터 허브 설정 화면까지 한국어/영어 모두 지원해요.
C
📷 허브 폰 카드
아기 방에 고정해두는 카메라 역할의 폰이에요. 카드를 누르면 안내 팝업이 먼저 뜨고, 확인하면 허브 설정 화면으로 넘어가요.
D
📱 뷰어 폰 카드
부모님이 들고 다니며 영상을 보는 폰이에요. 카드를 누르면 팝업 없이 바로 뷰어 화면으로 진입해요.
!
두 폰 모두 같은 Wi-Fi에 연결해야 해요
허브폰과 뷰어폰이 동일한 Wi-Fi 네트워크에 연결되어 있어야 영상이 전송돼요. 한 쪽이 모바일 데이터를 사용 중이면 연결이 되지 않아요.
💡 처음 사용 순서
허브폰: 허브 폰 카드 탭 → 팝업 확인 → 허브 설정 화면으로 이동
뷰어폰: 뷰어 폰 카드 탭 → 바로 뷰어 화면으로 이동

허브 설정 화면 (스크롤 가능)
A
화면 배경 & 헤더
어두운 갈색 배경이에요. 상단에 "🌙 허브 폰 설정" 제목과 우측에 "나가기" 버튼이 있어요. 나가기를 누르면 역할 선택 화면으로 돌아가요.
B
가로 모드 권장 배너 (주황색)
허브폰은 가로 모드로 사용해야 해요. 주황색 배너로 가로로 놓으라는 안내가 표시돼요. 세로 모드에서 방송 시작 시 회전 경고 오버레이가 뜨고 작동이 안 돼요.
C
1단계. 기기 연결 토큰
두 개의 코드 박스(A코드 + B코드)가 표시돼요. 이 코드를 뷰어폰에 입력해 연결해요. 옆의 🔄 버튼으로 코드를 새로 발급받을 수 있어요. 코드는 72시간 동안 방송이 없으면 만료돼요.
D
2단계. 녹화 영상 보관
이벤트(울음·얼굴 이탈) 감지 시 15초 영상이 기기에 자동 저장돼요. 저장소 사용량 바가 표시되고, 📂 버튼으로 저장된 클립을 바로 볼 수 있어요.
E
3단계. 화질 설정
고화질(1080p)과 절전 모드(720p) 중 선택해요. 기본값은 절전 모드예요. 기기가 뜨거워지거나 구형이면 절전 모드를 권장해요.
F
4단계. 마법 센서 (선택)
오울리 블루투스 온습도 센서(LYWSD03MMC)를 연결할 수 있어요. 선택 사항이에요 — 없어도 앱 사용에 지장 없어요.
G
설정 완료 버튼 (주황-황금 그라디언트)
모든 설정 확인 후 이 버튼을 눌러야 다음 단계(ROI 관심 영역 설정)로 넘어가요. 03b페이지에서 이어서 설명해요.
💡 Tip  코드는 한 번 발급받으면 뷰어폰에 등록해두면 돼요. 매번 새로 입력할 필요 없이, 허브폰이 켜져 있으면 자동으로 연결을 시도해요.

허브폰 설정 화면에 표시된 코드 2개(각 4자리)를 순서대로 입력해요. 이름은 방장이 누가 접속하는지 알 수 있도록 입력하는 거예요 (예: 할머니, 삼촌).

B

C
코드를 여러 번 틀리면?
틀린 횟수에 따라 접속이 일시 제한돼요. 3단계까지 있어요: 1단계(일시 제한) → 2단계(24시간 차단) → 3단계(30일 차단). 정확한 코드를 허브폰에서 확인하고 입력해 주세요.

알림을 받으려면 뷰어폰의 알림 권한이 허용되어 있어야 해요

알림 설정 · 가족 관리 (스크롤)
🧠 AI 스마트 감지
울음 감지와 수면 안전(얼굴) 감지를 각각 켜고 끌 수 있어요. 울음 감지는 dB 기준을, 수면 감지는 얼굴이 감지 안 될 때 알림을 보내기까지의 대기 시간(초)을 슬라이더로 조절해요. 위 화면처럼 수면 안전 감지 스위치를 끄면 얼굴 이탈 알림이 오지 않아요.
알림이 너무 잦으면 dB 기준을 높이거나 대기 시간을 늘려보세요
🌡️ 환경 센서 알림 기준
블루투스 온습도 센서가 연결되어 있을 때만 사용할 수 있어요. 방 온도와 습도의 적정 범위를 설정하면, 범위를 벗어날 때 알림이 와요. 센서가 없으면 흐릿하게 표시돼요.
🔔 기기 알림 및 소리
백그라운드 알림을 켜두면 뷰어폰 화면이 꺼져 있어도 울음·얼굴 알림을 받을 수 있어요. 알림 볼륨 슬라이더로 알림음 크기를 조절할 수 있어요.
🔒 보안 및 연결 관리
접속 코드를 복사하거나 가족에게 초대 링크를 보낼 수 있어요. 가족 명단에서 각 멤버의 알림 수신 여부를 켜고 끌 수 있고, 멤버 이름(호칭)도 직접 수정할 수 있어요.
입장 대기 중인 가족이 있으면 이 섹션에서도 승인·거절할 수 있어요
설정 탭 아이콘에 빨간 점이 생기면 대기 중인 가족이 있다는 신호예요
